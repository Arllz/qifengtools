单例模式：
1.一个类只有一个对象。
2.必须是自动创建这个类的对象。
3.整个对象只有这一个对象。
具体实现角度来讲，需要保证以下三点：
1.单例模式的类只提供私有的构造函数。
2.类定义中含有一个该类的静态自有属性。
3.该类提供了一个静态的公有函数用于创建或者获取他本身的静态私有对象。
4.该类需要有一个私有的clone方法，防止被克隆。

注意：单例模式不能用关键字new实例化对象。

php单例模式的实例：
class Single{
    static private $instance;//私有的静态属性用以保存对象
    private $config;
	
	//私有属性的构造方法，防止被new
	private __construct($config)
	{
	    $this->config = $config;
	}
	
	//私有属性的克隆方法，防止被new
	private function __clone(){}
	
	//静态方法 用以实例化调用
	static public function instance($config)
	{
	    if (!self::$instance instanceof self)
		{
		    self::$instance = new self($config)
		}
		return self::$instance;
	}
}

//初始化对象
$single_case = Single::instance("测试实例化");
